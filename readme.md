🚀 AI Resume Analyzer

«Built an AI-powered system that evaluates resumes against job descriptions and gives actionable insights.»

---

🔍 What this project does

Instead of blindly applying to jobs, this tool:

- Compares your resume with a job description
- Gives a match score (%)
- Highlights strengths & missing skills
- Suggests how to improve your profile

---

🧠 Why I built this

Most students (including me) don’t know:

- Why they’re getting rejected
- What skills they’re missing

So I built this to simulate how recruiters evaluate resumes using AI.

---

⚙️ Tech Stack

- Frontend → React (Vite)
- Backend → Node.js, Express
- AI → Groq API
- Auth → JWT
